# AI agents library

This repository contains modular AI agents and reusable skills
for UX, interaction, and product design work.

## How to use
1. Select an agent from `/agents`
2. Select one or more skills from `/skills`
3. Paste links into your AI tool
4. Provide problem context

## Design principles
- Agents define responsibility
- Skills define capability
- Markdown only
- No hidden logic
